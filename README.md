> 侵权请发 issue 联系删除，谢谢 :)

每月🦌管签到插件。Mioku/TypeScript 版

## ✨ 功能

- 自🦌：每天签到
- 帮🦌：帮别人签到
- 补🦌：补本月之前漏签的日子
- 🦌历：查看自己/别人的本月🦌签到日历
- 🦌榜：查看本群本月🦌签到排行榜
- 帮🦌 on/off：允许/禁止被别人帮🦌
- 禁🦌：管理员禁止某人在一段时间内🦌

所有命令中的「🦌」都可以用「鹿」字代替。

## 🎉 指令

| 指令                | 说明                  | 权限   |
|-------------------|---------------------|------|
| `🦌` / `鹿`        | 自🦌一次               | 全员   |
| `🦌 @某人`          | 帮某人🦌一次（仅群组）        | 全员   |
| `补🦌 <日>`         | 补签本月某天              | 全员   |
| `🦌历` / `🦌历 @某人` | 查看本月🦌日历            | 全员   |
| `🦌榜`             | 查看本月本群🦌排行榜（仅群组）    | 全员   |
| `帮🦌 on/off`      | 允许/禁止别人帮自己🦌        | 全员   |
| `帮🦌 on/off @某人`  | 允许/禁止帮某人🦌          | 群管理员 |
| `禁🦌 @某人 [时长]`    | 禁止某人🦌一段时间，省略时长视为解禁 | 群管理员 |

时长支持 `30s` / `5m` / `2h` / `1d` / `1w` 等写法，最长 30 天。
